Data Dictionary
Overview

This document describes the main columns used throughout the project.

unique_key – Unique identifier for each 311 service request.
created_date – Date and time the request was created.
closed_date – Date and time the request was closed.
agency – City agency responsible for handling the request.
complaint_type – Main category of the reported issue.
descriptor – More specific information about the complaint.
borough – NYC borough where the request was reported.
incident_zip – ZIP code of the reported location.
resolution_time_hours – Time taken to resolve the request, measured in hours.
resolution_time_days – Time taken to resolve the request, measured in days.
created_month – Numeric month used for monthly analysis.
created_month_name – Month name used for reporting and trend analysis.
created_day_of_week – Day of the week the request was created.
